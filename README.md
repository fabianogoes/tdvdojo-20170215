# TDV Dojo

### Setup

**install NodeJs**   

```shell
$ sudo apt-get install nodejs
$ sudo apt-get install npm

$ nodejs -v && npm -v
```   

**install dependences Front-End by nodejs**   

```shell
$ npm install angular@1.2.28 --save
$ npm install bootstrap --save
```
